## 功能

*以下功能仅供测试/调试使用*

- Simple DICOM Dump [**慎用修改和保存功能**]
- Worklist SCU/SCP with MPPS
- Query Retrieve SCU/C-MOVE SCU
- C-STORE SCU/SCP
- Print SCU/SCP
